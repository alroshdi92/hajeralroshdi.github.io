2. **Clone your repository:**
   - Open GitHub Desktop
   - File ‚Üí Clone Repository
   - Select `hajeralroshdi.github.io`
   - Choose a local path (e.g., Desktop)

3. **Copy all files** from `C:\Users\hajer.r\Desktop\CV\` to the cloned repository folder

4. **Commit and push:**
